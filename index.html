Media Queries

(Width: 1025+)

With media queries we can target device specific information and create css styles to optimize our site.

One of the pieces of information we can use is the size of the device screen that is viewing our page.

We can target specific device widths and have our page react differently depending on the size.

(Width: 1024-721)

We can target this specific information by utilizing @media rule in our css file.

@media (max-device-width: 1024px) {
          body {
            background-color: black;
            color: white;
          }
        }

You can have multiple instances of the @media rule so that you can target a wide variety of device sizes...

(Width: 720 and lower)

We can also target height as well...
